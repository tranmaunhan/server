Deploy Manager

Build Once, Run Anywhere: image của Deploy Manager giống nhau trên mọi server. Mọi khác biệt nằm ở `.env`, `docker-compose.yml`, whitelist app và route nginx.
Service allowlist · Nginx UI · Bearer Token

GitHub Actions chỉ gửi `serviceName`, còn lại để Deploy Manager kiểm soát.

Hệ thống chỉ cho phép `docker compose pull` và `docker compose up -d` trên những service có trong whitelist. Sau khi deploy xong, service sẽ health check, ghi audit log và có thể reload nginx từ UI.
